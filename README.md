# KotKotlin

> Repositório dedicado aos meus estudos em Kotlin.

## 📚 Sobre

O **KotKotlin** é um repositório criado para centralizar meus estudos e experimentações com a linguagem **Kotlin**.

O objetivo é evoluir continuamente por meio do desenvolvimento de projetos de diferentes níveis de complexidade, explorando desde os fundamentos da linguagem até conceitos mais avançados de arquitetura, design e boas práticas de desenvolvimento.

Ao longo deste repositório, busco aplicar conhecimentos como:

* Desenvolvimento de APIs e aplicações utilizando Kotlin;
* Clean Code e boas práticas;
* Princípios SOLID;
* Testes unitários e de integração;
* Arquitetura em Camadas (Layered Architecture);
* Clean Architecture;
* Arquitetura Hexagonal (Ports & Adapters);
* Padrões de Projeto (Design Patterns);
* Desenvolvimento orientado a domínio (DDD), quando aplicável.

Cada projeto possui um propósito específico de aprendizado, permitindo explorar diferentes abordagens e comparar soluções para um mesmo problema.

## 🚀 Objetivo

Construir uma base sólida em Kotlin através da prática, documentando a evolução do aprendizado e aplicando conceitos utilizados em projetos reais de desenvolvimento de software.
